How to Use
Clone this repository or download the source code.
Ensure you have Python 3.11 installed on your system.
Install the required packages using pip: pip install numpy matplotlib pillow
Run the application: python polynomial_approximation_gui.py
Use the GUI to add data points, specify the degree of the polynomial, and visualize the approximation.
Close the application by pressing Shift + Enter.


Overview
This Python application provides a graphical user interface (GUI) for polynomial approximation. Users can input data points, specify the degree of the polynomial, and visualize the polynomial approximation.

Features
Add data points with x and y coordinates.
Choose the degree of the polynomial for approximation.
Visualize the data points and the polynomial approximation.
Reset the input data and graphs.
Close the application using a keyboard shortcut.
